# Virgobeta
It is an freenode IRC bot written in Python.

* Its prefix is '&'
* It has 5 commands.
  * hi
    - Greets you or someone.
  * help
    - Runs the help module.
  * bf
    - The brainf*** interpreter.
  * ul
    - The underload interpreter. Not implemented.
  * unlambda
    - The unlambda interpreter. Not implemented.
